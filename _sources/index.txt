=========
Asterisms
=========

*View the night sky as ancient man did.*

The goal of this project is to facilitate the logging and cartography of historical and cultural asterisms and when particularly ancient, providing the functions to view them as ancient man would.

The documentation is available through the links in the Table of Contents below.

You can also visit:

* Official `Python Package Index <https://pypi.python.org/pypi/asterisms>`_ entry
* GitHub `project page <https://github.com/digitalvapor/asterisms/>`_
* GitHub `issue tracker <https://github.com/digitalvapor/asterisms/issues>`_
